== Description ==

= Inbox Form WordPress plugin =

The Official Inbox Form plugin makes it easy to grow your newsletter subscriber list from your WordPress blog or website. The plugin automatically integrates your Wordpress form with your Inbox email marketing account.

If you don't have Inbox account yet, [you can signup for a FREE here](https://useinbox.com/).

Once you activate the plugin, you’ll be able to select and add any of the pre-built forms from your Inbox account. You can place the form in the sidebar using a widget or use a shortcode to put it wherever you want.

Setup is fast and easy! You just need to enter your Inbox account API Key and you’re all set.

Plugin features include:

* Easily-to-add inline & popup forms from Inbox to your WordPress blog or website
* Wordpress 5 new editor support
* Place forms using widget or shortcode
* Auto place popup forms by rules

== Installation ==

= Method 1 =

1. Login to your WordPress admin panel.
2. Open Plugins in the left sidebar, click Add New, and search for Inbox plugin.
3. Install the plugin and activate it.

= Method 2 =

1. Download the Inbox plugin.
2. Unzip the downloaded file and upload to your /wp-content/plugins/ folder.
3. Activate the plugin in Wordpress admin panel.

= Setup =

1. After successful installation you will see Inbox icon on the left sidebar. Click it.
2. Enter your Inbox account API key. You can find it in your Inbox account by clicking "API Keys" link in the left sidebar.
3. Click "Forms" link under at Inbox menu on the left sidebar. You can add new forms or edit forms from this page.
5. If you want to include form in the sidebar of your blog or website, go to Appearance > Widgets and drag "Inbox Form" to the sidebar. Choose which signup form to display.
6. If you want to include form inside your post or page, use shortcodes. You will see Inbox icon in your content editor, click it and choose which form to display. It will put a shortcode (for example [inbox_form form_id=1]) and your visitors will see form in that place or popup.
